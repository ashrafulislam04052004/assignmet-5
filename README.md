1️⃣ What is the difference between var, let, and const?
  ans:
*var → older, less used
*let → value can be changed
*const → value cannot be changed


2️⃣ What is the spread operator (...)?
  ans: ... can be used to easily copy or combine data from an array or object.


3️⃣ What is the difference between map(), filter(), and forEach()?
   ans: 
*map() → creates a new array by modifying it.
*filter() → selects elements according to conditions.
*forEach() → just runs the loop, doesn't return anything.


4️⃣ What is an arrow function?
ans:
*This is a method of writing a short, concise function.
*The function keyword is not used.
*Syntax: (param) => { /* code */ }


5️⃣ What are template literals?
ans: This is an easy way to write strings and insert variables or expressions.
